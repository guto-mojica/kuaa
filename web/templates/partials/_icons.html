{#- Offline, inline-SVG icon macro. Replaces the external Phosphor web component that was loaded from the unpkg CDN — the project must run with ZERO external network references. Path data is the official Phosphor "regular" set (MIT, vendored verbatim under web/static/icons/phosphor/ with the upstream LICENSE for provenance); it is embedded here so rendering needs no runtime file IO and the asset set is deterministic. Parity notes: * viewBox 0 0 256 256 + fill="currentColor" => color follows the surrounding text color exactly as the old web component did. * class "ph-icon" sizes the glyph at 1em (see main.css), matching the old inline-font sizing; callers that need a larger glyph (the about-modal logo) get an extra class whose CSS font-size rule still applies because width/height are 1em-relative. The `extra` arg carries through state classes (e.g. "spin", "modal-logo") the old usages set via `class=`. -#} {%- macro icon(name, extra="") -%} {%- endmacro -%}
