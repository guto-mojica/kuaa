{#- About modal — Mojica redesign (Task 29). Atmospheric backdrop (blurred keyframe mosaic + vignette + tinted grain) + dim overlay + 660-wide modal panel with: * header — brand mark, runtime tag, title, version row, lede; * stats — 4-cell strip (films · scenes · runtime · time range); * models — attribution cards for every model in the pipeline; * stack — coloured pills for the runtime stack; * credits — institutional credit grid; * footer — repo / docs links + close button. The data is built by ``api.services.about_service.build_about_context`` and exposed on the context as ``version``, ``build_sha``, ``build_date``, ``stats``, ``models``, ``stack``, ``credits``, ``mosaic_keyframes``. See that module's docstring for the shape. Close behaviour: clicking the overlay (NOT the modal), the X button, the footer "Close" button, or pressing Esc empties ``#modal-container``, matching the legacy modal's behaviour and the htmx swap target on the About link in base.html. The close logic is hosted in an Alpine ``x-data`` scope on the outer ``.ab-app`` element: * ``close()`` empties ``#modal-container``, which removes this modal from the DOM (and on /about, the page's own listener notices the empty container and bounces to "/"). * ``@click.self`` on .ab-overlay replaces the previous ``onclick="event.stopPropagation()"`` on .ab-modal — clicks on the modal target a descendant, never .ab-overlay itself, so ``.self`` filters them out without the stopPropagation hack. * ``@keydown.escape.window`` adds the Esc shortcut the X button's tooltip already promises; the old inline-onclick path never had a key handler. -#} {% from "partials/_icons.html" import icon %}
